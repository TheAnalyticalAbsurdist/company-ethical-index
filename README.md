# Company Ethical Index

**Ethical Resonance Score (ERS) for organizations** – six‑dimensional assessment, migration pathways.

## 📊 What Is ERS?
A 0‑100 score measuring how well a company resonates with the 66‑band wavelength framework across six dimensions:

1. **Environmental** – resource use, emissions, biodiversity impact.
2. **Labor** – wages, safety, empowerment, growth opportunities.
3. **Governance** – transparency, accountability, stakeholder voice.
4. **Social** – community engagement, equity, inclusion.
5. **Supply‑Chain** – ethical sourcing, circularity, partner alignment.
6. **Innovation** – R&D investment, openness, future‑readiness.
